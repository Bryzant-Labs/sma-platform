# SMA Evidence Graph

Open-source evidence graph for Spinal Muscular Atrophy (SMA) drug research.

## Description

This dataset contains structured biological evidence for SMA research, including:
- **748 papers** from PubMed
- **2,098 claims** extracted via LLM analysis
- **10 therapeutic targets** (SMN1, SMN2, STMN2, PLS3, NCALD, UBA1, CORO1C, etc.)
- **7 drugs** (nusinersen, risdiplam, onasemnogene, apitegromab, etc.)
- **449 clinical trials** from ClinicalTrials.gov
- **7 omics datasets** from GEO
- **5 hypotheses** synthesized from evidence

## Data Files

| File | Description | Rows |
|------|-------------|------|
| data/targets.json | SMA therapeutic targets | 10 |
| data/sources.json | PubMed papers with abstracts | 748 |
| data/claims.json | LLM-extracted claims from abstracts | 2,098 |
| data/evidence.json | Claim-source evidence links | 2,098 |
| data/drugs.json | Approved and pipeline SMA drugs | 7 |

| data/graph_edges.json | Knowledge graph edges | 0 (in progress) |

## Schema Notes

### sources.json
Each entry is a PubMed paper with fields: `pmid`, `title`, `abstract`, `authors`, `journal`, `pub_date`, `doi`.

### claims.json
LLM-extracted structured claims from paper abstracts. Fields: `claim_id`, `source_id`, `claim_text`, `claim_type`, `confidence`, `target_id`, `drug_id`.

### evidence.json
Links claims to sources with evidence metadata. Fields: `evidence_id`, `claim_id`, `source_id`, `evidence_type`, `strength`.

### trials.json
Clinical trials from ClinicalTrials.gov. Fields: `nct_id`, `title`, `status`, `phase`, `drug`, `start_date`, `completion_date`, `enrollment`.

### hypotheses.json
Synthesized research hypotheses with supporting evidence summaries and confidence scores.
